This package includes examples of Netbeans client applications.
  • The {@link vimoir.examples.Phonemic} client is an example of a {@link vimoir.netbeans.NetbeansEventHandler}. The Phonemic client uses the phonemic java library to make Vim speak the content of buffers or speak operational messages such as Netbeans events.
  • The {@link vimoir.examples.Process} client allows to start tasks from Vim in the background, and to get the results of the process execution in a Vim balloon when the process terminates.
